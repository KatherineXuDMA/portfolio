Michaelsoft Paint

Welcome to Michaelsoft Paint! This is a simple retro-themed drawing program where you can draw using the mouse and interact with a set of keys.

CONTROLS:
1 - Black Brush     6 - Green Brush
2 - Sky Blue Brush  7 - Blue Brush
3 - Pink Brush     8 - Rainbow brush
4 - Red Brush     9 - Cloud Brush
5 - Orange Brush
q - Cat Brush
w - Star Brush
0 - Erase
x - Reset Canvas   k - Increase Brush Size by 1
p - Print Canvas   j - Decrease Brush Size by 1
